# Elevation

## Description

Elevation represents the height of the terrain above mean sea level. It is a fundamental topographic parameter derived from the Digital Elevation Model (DEM) and influences climatic conditions, vegetation distribution, and geomorphological processes.

---

## Elevation (DEM)

Elevation data was obtained from a Digital Elevation Model (DEM), where each pixel stores the elevation value in meters.

### Processing

```
Load DEM → Clip to Study Area → Reproject to UTM (EPSG:32644)
```

Output file: `utm_Elevation.tif`

### Role in Study

Elevation plays an important role in controlling terrain characteristics and environmental conditions that influence landslide occurrence.

- It affects slope formation and terrain morphology.
- It influences vegetation distribution and soil development.
- It reflects variations in climatic conditions such as precipitation and temperature.

These factors collectively contribute to slope stability and landslide susceptibility.
