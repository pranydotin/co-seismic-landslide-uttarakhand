# Slope

## Description

Slope represents the steepness of the terrain and is calculated as the rate of change in elevation between neighboring cells. Higher slope values generally indicate steeper terrain, which is more susceptible to landslides due to gravitational forces acting on slope materials.

---

## Processing

Slope was derived from the DEM using terrain analysis tools:

```
Terrain Analysis → Morphometry → Slope, Aspect, Curvature
```

Output file: `slope.tif`

---

## Role in Study

Slope represents the steepness of the terrain. Areas with steeper slopes are generally more susceptible to landslides due to increased gravitational forces acting on soil and rock materials.
