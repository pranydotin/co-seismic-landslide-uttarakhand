# LSI Map Generation

## Overview

The Landslide Susceptibility Index (LSI) map was generated by summing all 13 FR-reclassified rasters. Each pixel in the LSI map represents the cumulative landslide susceptibility based on the combined contribution of all conditioning factors. The LSI is a continuous surface — higher values indicate greater environmental similarity to known landslide locations.

---

## Formula

$$LSI = \sum_{j=1}^{13} FR_j$$

Where $FR_j$ is the FR-reclassified raster for conditioning factor $j$.

---

## Tool

**QGIS** — Raster Calculator  
`Raster → Raster Calculator`

> Python-based summation was attempted but caused session crashes due to the large spatial extent of LULC and soil type rasters. QGIS Raster Calculator was used instead as it processes rasters row-by-row internally, avoiding memory overflow.

## Processing Steps

### Step 1 — Load FR Rasters in QGIS

Load all 13 `FR_*.tif` files into QGIS as raster layers.

### Step 2 — Open Raster Calculator

```
Raster → Raster Calculator
```

### Step 3 — Enter Expression

```
"FR_slope_class@1" +
"FR_utm_Aspect_class@1" +
"FR_elevation_class@1" +
"FR_plan_curv_class@1" +
"FR_profile_curv_class@1" +
"FR_twi_class@1" +
"FR_tri_class@1" +
"FR_ndvi_class@1" +
"FR_utm_lulc@1" +
"FR_proximity_roads_class@1" +
"FR_proximity_rivers_class@1" +
"FR_proximity_fault_class@1" +
"FR_utm_soil_type@1"
```

### Step 4 — Set Output Parameters

| Parameter | Value |
|-----------|-------|
| Output CRS | UTM Zone 44N (EPSG:32644) |
| Output format | GeoTIFF |
| Output file | `LSI_map.tif` |

### Step 5 — Run and Verify

After generation, verify the LSI map by:
- Checking the value range (should be a continuous positive surface)
- Visually inspecting spatial patterns — high LSI values should concentrate along roads, rivers, and steep terrain consistent with known landslide distributions in Uttarakhand

---

## Output

| File | Description |
|------|-------------|
| `LSI_map.tif` | Continuous LSI surface — sum of all 13 FR rasters |

**Value range:** Continuous positive values representing cumulative FR contribution per pixel  
**Higher LSI** → greater environmental similarity to landslide conditions  
**Lower LSI** → less similar to landslide conditions

---

## Next Step

The LSI map is classified into **5 similarity zones** using Natural Breaks (Jenks) classification in QGIS, producing `LSI_zones.tif`. These zones form the basis for the optimized non-landslide point selection strategy.

---

## Notes

- All 13 input FR rasters must share identical extent, resolution, and CRS before summation
- NoData pixels (value = -9999) in any input raster propagate as NoData in the LSI output
- Processing time approximately 5–15 minutes depending on system RAM
